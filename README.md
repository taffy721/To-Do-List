# To-Do-List
 I just learnt how to read files using python and so I created a to do list. What makes this unique is that your tasks will be stored into a txt file so you can save it and view it later. 

How to use:
First make sure you have either created a txt file or clone the one provided in this repository.
Next copy the file path for the txt file.
Past the file path in the code where it in the string on the first line.
Finally run the programme.
